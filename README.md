# Mascot-of-Chat2Job
Taby is the AI mascot of Chat2Job.
More than an avatar, Taby is designed to be a living AI companion that represents conversations, interviews, recruiting workflows, and human-AI collaboration inside the Chat2Job ecosystem.

Vision

Most software products have logos.

Chat2Job has Taby.

Taby is designed to communicate emotion, attention, thinking, listening, waiting, and interaction through simple expressive animations.

Features
Pure SVG implementation
No external dependencies
Lightweight and embeddable
Multiple emotional states
Smooth spring-based animations
Fully browser-native
